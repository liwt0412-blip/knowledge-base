# 常用快捷键

| 快捷键                 | 功能                     |
| ---------------------- | ------------------------ |
| ctrl + p               | 快速查看创建对象时的属性 |
| ctrl + r               | 关键字替换               |
| ctrl + i               | 调出接口方法             |
| ctrl + o               | 调出重写方法             |
| ctrl + n               | 搜索类名                 |
| ctrl + shift + u       | 变量切换大小写           |
| ctrl + shift + r       | 全局搜索                 |
| ctrl + shift + -       | 全部折叠                 |
| ctrl + shift + alt + j | 变量名直接修改           |
| ctrl + alt + m         | 将代码抽取为方法         |
| ctrl + alt + v         | 快速抽取（反链式）       |
| ctrl + alt + t         | 快速调出try-catch        |
| ctrl + alt + left      | 快速定位到上一个类       |
| ctrl + alt + right     | 快速定位到下一个类       |
| alt + F7               | 查看当前类有哪些方法     |
| alt + insert           | 快速生成该类文件的测试类 |
