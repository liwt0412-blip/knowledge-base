# Typora 新手完整实用技巧

先解决你两个疑问：

1. 导航栏看不到字体颜色、高亮、彩色文字

   Typora 是

   极简模式

   ，可视化按钮很少，

   靠简单代码 / 语法

   实现变色、引用、高亮，不是点按钮。

2. 引用、分割线、列表、代码块全都有，都是 Markdown 基础语法，超好记。

------

## 一、你最想要的：文字变色 / 字体颜色 / 高亮

### 1. 文字改颜色（任意颜色）

```html
<span style="color:red">红色文字</span>
<span style="color:blue">蓝色文字</span>
<span style="color:green">绿色文字</span>
```

效果：<span style="color:red">红色文字</span>

### 2. 文字背景高亮（黄色标记重点）

```html
<mark>这是高亮底色，用来画重点</mark>
```

效果：<mark>这是高亮底色，用来画重点</mark>

------

## 二、引用块（你看到别人的灰色引用框）

### 语法：开头加一个 `>` + 空格

```md
> 这是引用内容
> 多行引用直接换行继续写
```

效果：

> 这是引用内容
>
> 多行引用直接换行继续写

------

## 三、最常用基础排版（必记）

### 1. 标题（一键分级）

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

### 2. 加粗 / 斜体 / 删除线

```
**加粗文字**
*斜体文字*
~~删除线文字~~
```

### 3. 无序列表 / 有序列表

```
- 无序列表1
- 无序列表2

1. 有序列表一
2. 有序列表二
```

### 4. 分割线（页面隔开）

输入 `---` 或 `***` 回车

```
---
```

------

```md

```
