Openclaw必装的三个skills.如果你 Openclaw 安装了，但是不好用，可能你没安装下面三个 Skill 技能！
1，tavily-search 联网技能。不联网跟瞎子一样！先开眼看世界。
安装命令：clawhub install tavily-search
2，find-skills 主动找技能解决问题。别天天问你。因为你更不知道！
安装命令：clawhub install find-skills
3，proactiva-agent-1-2-4 主动代理 Agent ,自我迭代升级技能。
安装命令：clawhub install proactive-agent-1-2-4
嗯，有这几个，应该会好用多了吧？
