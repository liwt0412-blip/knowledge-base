# day06_Python⽂件与异常


## 1.函数的参数进阶

### 形参：在函数定义时，所编写的参数就称之为形式参数
实参：在函数调⽤时，所传递的参数就称之为实际参数

### 函数的参数类型(调⽤)

#### 位置参数

##### 理论上，在函数定义时，我们可以为其定义多个参数。但是在函数调⽤时，我们也应该传递多个参数，正常情况，其要⼀⼀对应。用 ，号隔开

###### def user_info(name, age, address)

#### 关键词参数（Python特有）

##### 函数调⽤，通过“键=值”形式加以指定。可以让函数更加清晰、容易使⽤，同时也清除了参数的顺序需求。

###### 调⽤函数（使⽤关键词参数）user_info(name='Tom', address='美国纽约', age=23)

####  函数定义时缺省参数（参数默认值）

##### 缺省参数也叫默认参数，⽤于定义函数，为参数提供默认值，调⽤函数时可不传该默认参数的值（注意：所有位置参数必须出现在默认参数前，包括函数定义和调⽤）。

###### def user_info(name, age, gender='男'):

### 不定⻓参数

#### 不定⻓参数也叫可变参数。⽤于不确定调⽤的时候会传递多少个参数(不传参也可以)的场景。此时，可⽤包裹(packing)位置参数，或者包裹关键字参数，来进⾏参数传递，会显得⾮常⽅便。

#### 不定⻓元组（位置）参数

##### def user_info(*args)

#### 不定⻓字典（关键字）参数

##### def user_info(**kwargs)

## 2.lambda函数

### 为什么要使⽤lambda：
1. 函数使⽤次数不多，就使⽤⼀次，不适合单独定义⼀个普通函数
2. 在调⽤⼀个函数时，这个函数需要传递的参数就是⼀个函数，作为参数进⾏传递的函数，不适合单独定义⼀个普通函数

### lambda表达式应⽤场景
• 如果⼀个函数有⼀个返回值
• 并且只有⼀句代码，可以使⽤ lambda简化。

### lambda表达式基本语法

#### 变量 = lambda 函数参数:表达式（函数代码 + return返回值）

##### # 求和
lambda_sum = lambda a,b: a + b


#### 定义⼀个函数，经过⼀系列操作，最终返回100

##### # 定义⼀个lambda函数
lambda_fn1 = lambda: 100
res = lambda_fn1()
print(res)

#### 带默认参数的lambda表达式

#### 带if判断（三⽬运算符）的lambda表达式

##### # 1 if name == "张三" else 0 如果 if 成⽴，那么返回1，不成⽴返回0
lambda_fun1 = lambda name: 1 if name == "张三" else 0

#### 列表数据+字典数据排序（重点）

##### # 给定以下⼀个列表:
student_list = [
{"name": "张三", "age": 20, "address": '南京'},
{"name": "李四", "age": 25, "address": '北京'},
{"name": "王五", "age": 23, "address": '上海'},
{"name": "赵六", "age": 19, "address": '⼴州'},
{"name": "⽥七", "age": 22, "address": '深圳'}
]
# 需求: 请对列表进⾏排序: 要求根据字典中年龄 从 ⼤到 ⼩ 排序
# 列表.sort(reverse=True) : 实现对列表中的数据进⾏排序
student_list.sort(key=lambda a: a['age'], reverse=True)
print(student_list)

## 3.⽂件的概念

### 内存中存放的数据在计算机关机后就会消失。要⻓久保存数据，就要使⽤硬盘、光盘、U 盘等设备。为了便于数据的管理和检索，引⼊了“⽂件”的概念。

#### ⼀篇⽂章、⼀段视频、⼀个可执⾏程序，都可以被保存为⼀个⽂件，并赋予⼀个⽂件名。操作系统以⽂件为单位管理磁盘中的数据。⼀般来说，⽂件可分为⽂本⽂件、视频⽂件、⾳频⽂件、图像⽂件、可执⾏⽂件等多种类别。

### ⽂件操作包含哪些内容呢？

#### 在⽇常操作中，我们对⽂件的主要操作：创建⽂件、打开⽂件、⽂件读写、⽂件备份等等

### open函数打开⽂件

#### f = open(name, mode)

##### name：是要打开的⽬标⽂件名的字符串(可以包含⽂件所在的具体路径)。
mode：设置打开⽂件的模式(访问模式)：只读r、写⼊w、追加a等。

#### ⽂件路径：① 绝对路径 ② 相对路径

##### ① 绝对路径：绝对路径表⽰绝对概念，⼀般都是从盘符开始，然后⼀级⼀级向下查找（不能越级），直到找到我们要访问的⽂件即可。

##### ② ==相对路径==：相对路径表⽰相对概念，不需要从盘符开始，⾸先需要找到⼀个参考点（就是
Python⽂件本⾝）

### 案例

#### # 写⼊abcd到当前⽬录下
f = open("./a.txt","w")
f.write("abcd")
f.close()


##### 强调⼀下：中⽂乱码问题，默认情况下，计算机常⽤编码ASCII、GBK、UTF-8

### ⽂件的读取操作

#### read(size)⽅法 ：主要⽤于⽂本类型或者⼆进制⽂件（图⽚、⾳频、视频...）数据的读取size表⽰要从⽂件中读取的数据的⻓度（单位是字符/字节），如果没有传⼊size，那么就表⽰读取⽂件中所有的数据。


#### readlines()⽅法 ：主要⽤于⽂本类型数据的读取
readlines 可以按照⾏的⽅式把整个⽂件中的内容进⾏⼀次性读取，并且返回的是⼀个列表，其中每⼀⾏的数据为⼀个元素。

#### readline()⽅法 ：⼀次读取⼀⾏内容，每运⾏⼀次readline()函数，其就会将⽂件的指针向下移动⼀⾏

如图⽚、⾳频、视频等格式

###### try:
可能遇到的错误代码
except Exception as e:
print(e)
