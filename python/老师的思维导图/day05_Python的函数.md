## 2- Python的推导式

### 1.什么是推导式？

#### 用一行代码，快速加工指定容器每个元素，生成一个新的容器

### 2.使用场景？

#### 对原容器进行遍历，生成一个新容器
可以使用推导式简化操作

### 3.语法格式：
