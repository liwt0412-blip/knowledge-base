## 4- Python的基础语法

### 1.注释

#### 1.什么是注释？

##### 对代码解释说明的文字
提高代码的可读性

#### 2.如何添加注释？

##### 单行注释：语法格式
   # 解释说明的文字
   python代码

##### 多行注释：语法格式
    """
      解释说明的文字1
      解释说明的文字2     
    """

###### 多行注释：语法格式
    '''
      解释说明的文字1
      解释说明的文字2     
    '''

##### 快速添加/取消注释：
      ctrl  +  / 

### 2.pycharm快捷键

#### 快速注释： ctrl  +  /

#### 复制当前行：ctrl  +  d

#### 删除当前行：ctrl  +  x

#### 格式化代码：ctrl  +  alt  +  l  (快捷键冲突)

#### 撤销：     ctrl  +  z
反撤销：  shift  +  ctrl  +  z

#### 移动当前行代码：shift  +  alt   +  箭头

#### 保存：ctrl  +  s

#### ... ...

### 3.变量

#### 1.概念：

##### 标签 = 存储对象的地址
指向对象/数据，方便操作对象中的数据

#### 2.语法格式：

##### 变量名称  =  值

#### 3.注意事项：

##### 1.建议赋值符号=两边都有一个空格（规范）

##### 2.变量命名
    一定要遵循规则
    尽量遵循规范

###### 规则：
     只能使用数字，大小写字母，下划线
     不能使用数字开头
     不能使用关键字（被python官方使用的单词）
     区分大小写

###### 规范：
     见名知意
     大驼峰/小驼峰/下划线

### 4.数据类型

#### 1.数值类型：int /float
2.布尔类型：bool (True|False)
3.字符串：str
4.列表/元组/集合/字典

### 5.输入和输出

#### 输出

##### 格式：
    print()

##### 输出格式：
    print(f'字符串模板{变量1}' )

##### 转义符：
    \n
    \t

##### 设置结束符：
    print( 字符数据 ，end='\n' )  
    自行修改end的内容

#### 输入

##### 格式：
   接收 =  input('提示信息')

##### 注意事项：
     接收的数据类型全部都是字符串类型
     阻塞接收用户输入的数据
     建议书写提示信息，提高用户体验

### 6.数据类型转换

#### 新类型数据 = int(x) : 转整形
新类型数据 = float(x)：转小数/浮点型
新类型数据 = str(x)：转字符串/文本类型
