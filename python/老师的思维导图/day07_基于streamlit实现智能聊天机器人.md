## 0- python异常处理
【实操】

### 1. 什么是异常？

#### 非正常，程序中一般指程序发生的问题/错误

### 2. python异常处理机制？

#### 1.停止执行后续代码
2.收集异常信息：异常信息（原因/位置..）封装起来，封装到异常对象中
3.将异常对象向上抛
4.在项目最外层进行捕获异常对象
5.获取异常对象，解析异常相关信息
6.将解析出异常信息，输出到控制台

##### 飘红：有助于开发者定位问题/发现问题/解决问题

### 3.如何进行异常处理？

#### 语法格式：
      try：
            可能发生异常的代码1
            可能发生异常的代码2
            .... ....
      except : 
            发生异常需要执行的操作  
      else:
            没有发生异常需要执行的操作
      finally:
            无论是否发生异常，都需要执行的操作

##### except:    #  捕获匹配所有异常

##### except  Exception :   #  捕获匹配所有异常

##### except  ZeroDivisionError：  # 只捕获处理具体指定的异常

##### except  ZeroDivisionError  as  别名：  # 只捕获处理具体指定的异常
          通过别名获取/操作异常

##### except 代码块可以有多个，注意大的异常处理建议放到最后

except  Exception:
      # 
except  A：
      #  
except  B：
      #
except   C：
      #
