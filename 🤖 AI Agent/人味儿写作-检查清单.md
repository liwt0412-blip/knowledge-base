# 事后检查清单：你动过的地方有没有 AI 味儿

> 只扫你动过的句子，不是全文。拿清单去扫原文是本末倒置——原文的"问题"很可能是手迹。

## 一、意义拔高

- 把普通事实说成"标志着""见证了""体现了""彰显了""折射出"
- 句尾挂"-ing 式"补尾："……，体现了团队的深厚积累"
- 把局部话题拔到时代高度："这将重新定义……"

**改法：** 删掉拔高从句，只留事实。

## 二、宣传腔

- "璀璨""深厚底蕴""赋能""打造""致力于""匠心"
- "不容错过""令人惊叹""极致体验"

**改法：** 换成可验证的具体信息。"历史悠久"不如"建于 1987 年"。

## 三、句式套路

| 套路 | 特征 |
|---|---|
| "不是X，而是Y" | 最高频 AI 假深刻。一篇出现一次都要警惕 |
| 排比三连 | 硬凑三个一组。一个排比是修辞，连续排比是模式 |
| 同义词轮换 | 同一个主语换着叫。人会自然重复，AI 才怕重复 |
| 假范围 | "从 X 到 Y"但 X 和 Y 不在一条真实的轴上 |
| 格言公式 | "X 是 Y 的语言""X 是一面镜子"——还原成具体判断 |
| 短句轰炸 | "没有犹豫。没有退路。只有向前。"——一个短句是强调，一排是表演 |
| 自问自答 | "结果呢？惨烈。""最可怕的是什么？没人察觉。" |
| 假坦诚开场 | "说实话""讲真"之后跟一个平平无奇的观点 |

## 四、格式痕迹

- **破折号**：最高频 AI 信号。每一个都替换（句号分句 / 逗号紧凑插入 / 冒号引出解释）
- **加粗滥用**：机械加粗关键词，尤其是「**加粗开头**：后接解释」
- **emoji 装饰**：标题和要点前的 🚀💡✅
- **标题后废话垫场**：标题下面先来一句重复标题的话

## 五、语气痕迹

- **聊天残留**："希望这对你有帮助""需要我展开吗"
- **谄媚**："好问题！""你说得太对了！"
- **模糊归因**："有专家认为""业内人士指出"——说不出名字就没有来源
- **万能展望结尾**："未来可期""让我们拭目以待"
- **挑战与展望段**："尽管面临诸多挑战……但依然……"
- **签到式过渡**："让我们深入探讨""话不多说"——直接说事

## 六、填充与对冲

- "值得注意的是""需要指出的是"：删掉后句子照常成立
- "在某种程度上或许可能"：叠加对冲。一个"可能"够了
- "为了实现这一目标"→"为此"；"在……的情况下"→"如果"

## 不要误伤（重要）

单个信号不说明任何问题。以下都不是 AI 的证据：

- 语法完美、风格统一（很多人就是写得好）
- 一个破折号、一处排比、一个"然而"（人也用）
- 干巴巴的文风（AI 味儿是特定模式，不是泛泛的无聊）
- 书面语和口语混用（这往往恰恰是真人）

**判断标准是聚集：** 一个破折号什么都不是。破折号 + 排比三连 + "璀璨的画卷" + "总而言之"同时出现，才是供词。

## 人写的迹象（看到就保住）

- 具体到难以编造的细节（真实地址、奇怪的引用）
- 没解决的矛盾心情
- 长短不一的呼吸（忽长忽短的句子）
- 自我打断和插话
- 带年代和圈子印记的梗
- 作者能说出理由的用词选择

## 核对流程

1. 列出这次改动的所有位置
2. 逐处对照上面六组检查
3. 问一句：「这几处改动里，哪里还能看出是 AI 改的？」
4. 修掉，收工。**不要顺手再去"优化"没动过的句子**
