# Agent 上下文与记忆入口

> 适用：上下文长度、长期记忆、知识库结构、技能、工具调用、多 Agent 协作。

> 边界：本模块只处理 Agent 协作、上下文、记忆、技能和 MCP；RAG 理论进入 AI 大模型基础，Python 实现进入 AI大模型开发，Java 实现进入 SpringAI+AIGC应用。

## 核心策略：分层 + 渐进式披露

```text
长期上下文（稳定规则 + 路由）
  → 模块入口（一页摘要 + 子主题索引）
    → 专题笔记（完整知识、案例、排错记录）
      → 原始资料（仅在需要验证时读取）
```

- 上下文是有限注意力预算；目标是最小但足够的高信号信息。
- 先用文件路径、模块入口和触发词定位，再按需读取正文。
- 阶段结束压缩为“决策、状态、待办、入口”，而非保留全部对话和工具输出。
- 原始资料不是已验证事实；结论须经核验后沉淀到专题笔记。

## 何时写入哪里

| 内容 | 去向 |
|---|---|
| 稳定、跨任务反复影响决策的偏好或原则 | [[00-我的长期上下文]] |
| 某领域的完整规则、案例、排错 | 对应模块专题笔记 |
| 某项目接口、表结构、配置、实现进度 | `🖥 项目笔记/项目名` |
| 截图、网页原文、未验证材料 | `📥 原始资料（待处理）` |
