# GitHub 项目：宇宙无敌表达训练系统

> 评估时间：2026-07-19
> 仓库：[fxy2311-youyou/expression-trainer](https://github.com/fxy2311-youyou/expression-trainer)
> ⭐ 476 · 🍴 104 · MIT · 创建 2026-07-06（13天）

## 项目概述

本地桌面应用（Electron），实时语音识别 → 词库匹配 → AI 分析报告。对着麦克风说话，检测填充词（嗯/啊/那个）、犹豫词（可能/也许）、笼统词，给出精准替代建议，生成 6 维度分析报告。「离线 ASR + 本地词库 + LLM API」，隐私友好。

## 技术架构

```
┌─────────────────────────────────────────┐
│ Electron 主进程                          │
│  ├── Sherpa-ONNX（离线语音识别）         │
│  ├── 词库匹配（emotion-lexicon.json）    │
│  └── AI 反馈（多后端 HTTP API）           │
├─────────────────────────────────────────┤
│ 渲染进程（Chromium）                     │
│  ├── 全屏字幕显示                        │
│  ├── 实时统计面板                        │
│  └── 分析报告弹窗                        │
└─────────────────────────────────────────┘
```

三层都是已有轮子的组合：
1. **Sherpa-ONNX** — 离线流式语音识别，中文优化
2. **emotion-lexicon.json** — 基于大连理工情感词库 7 大类结构，27000 词
3. **LLM API** — 支持 DeepSeek/OpenAI/Ollama 多后端

## 词库说明

`data/emotion-lexicon.json` 包含：

| 分类 | 内容 |
|------|------|
| 情绪词 | 130+，7 大类（喜怒哀惧恶惊）× 强度 1-9 |
| 笼统词→精准词 | 25 组高频替代建议 |
| 填充词表 | 24 个常见口头禅（嗯、啊、那个、然后…） |
| 犹豫词表 | 19 个弱化表达（可能、也许、我觉得…） |
| 程度词梯度 | 弱→中→强→极 四级 |
| 画面化描述 | 10 组「抽象→具象」转换 |
| 犹豫→直接 | 8 组对照示例 |

## 对 liwt 的价值判断

| 角度 | 判断 | 说明 |
|------|------|------|
| **面试练习** | ✅ 直接能用 | 实时纠偏填充词/犹豫词，比事后读修正记录强 |
| **架构学习** | ❌ 不值得学 | Electron + 词表匹配 + LLM Prompt，无迁移价值 |
| **工程参考** | ⚠️ 词库分类值得看 | `emotion-lexicon.json` 分类体系有参考价值 |
| **抖音素材** | ⚠️ 一般 | 有点话题性，不够硬核 |

## 诚实评价

**技术含量中等偏低。** 实质是「语音识别 SDK + 词典匹配 + LLM Prompt 工程」，三块都是调用已有轮子。476 星更多是因为工具实用 + 刚需场景，不是技术创新。

**但产品价值是实的。** 离线 ASR 意味着无需联网、隐私友好。词库 27000 条足够细，多 LLM 后端兼容做得好。这是一个「做得好的工具」，不是一个「值得学的框架」。

## WSL 集成可行性

| 层面 | 可行性 | 原因 |
|------|--------|------|
| Electron 桌面应用 | ❌ | GUI 应用，WSL 跑不了 |

| 离线处理音频文件 | ✅ | 录音后喂 `.wav` → 转写 → 分析 |
