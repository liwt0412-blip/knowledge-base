# Java高级

# 一、Arrays 常用 API 速查表

| 方法                                    | 作用说明                           |
| :-------------------------------------- | :--------------------------------- |
| `sort(数组)`                            | 对数组**默认升序排序**             |
| `sort(数组, 比较器)`                    | 自定义规则排序（引用类型数组）     |
| `binarySearch(数组, 元素)`              | **二分查找**，前提：数组必须先有序 |
| `copyOf(数组, 新长度)`                  | 数组拷贝，可扩容 / 缩容            |
| `copyOfRange(数组, 起始索引, 结束索引)` | 区间拷贝 `[start,end)`             |
| `fill(数组, 元素)`                      | 用指定元素**填充整个数组**         |
| `toString(数组)`                        | 数组转字符串，方便打印查看         |
| `asList(T... a)`                        | 数组转 **List 集合**               |
| `equals(数组1, 数组2)`                  | 比较两个数组**内容是否完全一致**   |

# 三、System 类常用 API 表格总结

`java.lang.System` 都是**静态方法 / 常量**，直接 `System.xxx()` 调用。

## 1. 常用常量

| 常量         | 作用                                 |
| :----------- | :----------------------------------- |
| `System.in`  | 标准**输入流**，键盘录入             |
| `System.out` | 标准**输出流**，控制台打印           |
| `System.err` | 标准**错误输出流**，红色打印异常信息 |



## 2. 时间相关

| 方法                         | 作用                                   |
| :--------------------------- | :------------------------------------- |
| `System.currentTimeMillis()` | 获取当前时间**毫秒值**（时间戳）       |
| `System.nanoTime()`          | 获取当前纳秒时间，用于**计算程序耗时** |



## 3. 数组操作

| 方法          | 作用                                 |
| :------------ | :----------------------------------- |
| `arraycopy()` | 数组高效拷贝，底层原生方法，效率极高 |



## 4. 退出与垃圾回收

| 方法                      | 作用                                              |
| :------------------------ | :------------------------------------------------ |
| `System.exit(int status)` | 终止当前 JVM，**0 正常退出，非 0 异常退出**       |
| `System.gc()`             | 建议 JVM 执行**垃圾回收**（只是建议，不立即执行） |



## 5. 获取系统属性

| 方法                             | 作用                 |
| :------------------------------- | :------------------- |
| `System.getProperties()`         | 获取系统所有属性对象 |
| `System.getProperty(String key)` | 根据键获取系统属性值 |



### 5.1 常用系统属性 key

| key            | 含义           |
| :------------- | :------------- |
| `os.name`      | 操作系统名称   |
| `user.name`    | 系统用户名     |
| `user.dir`     | 当前项目根路径 |
| `java.version` | JDK 版本       |



## 6. 环境变量

| 方法                         | 作用                     |
| :--------------------------- | :----------------------- |
| `System.getenv()`            | 获取所有环境变量         |
| `System.getenv(String name)` | 根据名称获取指定环境变量 |



# 四、Exception 异常常用 API 表格总结

`Exception` 和父类 `Throwable` 是所有异常的根，**所有异常类都能用以下方法**。

## 1. Throwable / Exception 常用核心方法

| 方法                         | 作用说明                                       |
| :--------------------------- | :--------------------------------------------- |
| `getMessage()`               | 获取异常**简要描述信息**                       |
| `toString()`                 | 获取异常类型 + 异常信息                        |
| `printStackTrace()`          | 打印**完整异常堆栈信息**（类名、行号、调用链） |
| `getStackTrace()`            | 获取异常堆栈数组，可自己遍历处理               |
| `getCause()`                 | 获取**原始引发的异常**（异常链）               |
| `initCause(Throwable cause)` | 手动设置异常根源                               |



## 2. 异常构造方法（自定义异常常用）

| 构造方法                                 | 作用                |
| :--------------------------------------- | :------------------ |
| `Exception()`                            | 创建空异常对象      |
| `Exception(String msg)`                  | 带异常提示信息      |
| `Exception(String msg, Throwable cause)` | 带信息 + 带原始异常 |
| `Exception(Throwable cause)`             | 只封装原始异常      |



## 3. 受检异常 & 运行时异常 区分

| 类型           | 父类               | 特点                              |
| :------------- | :----------------- | :-------------------------------- |
| 编译时受检异常 | `Exception`        | 必须 `try-catch` 或 `throws` 声明 |
| 运行时异常     | `RuntimeException` | 无需强制捕获，代码不规范导致      |



## 4. 常见内置异常

| 异常类                            | 含义             |
| :-------------------------------- | :--------------- |
| `NullPointerException`            | 空指针异常       |
| `ArrayIndexOutOfBoundsException`  | 数组索引越界     |
| `StringIndexOutOfBoundsException` | 字符串索引越界   |
| `ClassCastException`              | 类型转换异常     |
| `ArithmeticException`             | 算术异常（除 0） |
| `IllegalArgumentException`        | 非法参数异常     |
| `IndexOutOfBoundsException`       | 索引越界父类     |



## 5. 关键字配套用法

| 关键字    | 作用                                 |
| :-------- | :----------------------------------- |
| `try`     | 包裹可能出异常的代码                 |
| `catch`   | 捕获并处理异常                       |
| `finally` | 无论是否异常**都会执行**（关闭资源） |
| `throw`   | 手动**抛出**单个异常对象             |
| `throws`  | 方法上**声明**可能抛出的异常         |



# 六、Stream 流常用 API 

Stream 分为**中间操作**（返回 Stream，可链式调用）、**终结操作**（关闭流，触发执行）

## 1. 获取 Stream 流（创建流）

| 方法                    | 作用                   |
| :---------------------- | :--------------------- |
| `集合.stream()`         | 普通集合获取**串行流** |
| `集合.parallelStream()` | 集合获取**并行流**     |
| `Arrays.stream(数组)`   | 数组转 Stream 流       |
| `Stream.of(多个元素)`   | 直接给元素创建流       |



## 2. 中间操作（延迟执行，返回 Stream）

| 常用方法             | 作用                                 |
| :------------------- | :----------------------------------- |
| `filter(Predicate)`  | **过滤**：过滤满足条件的元素         |
| `map(Function)`      | **映射转换**：元素类型 / 内容转换    |
| `distinct()`         | **去重**：去除重复元素               |
| `sorted()`           | 自然排序                             |
| `sorted(Comparator)` | 自定义比较器排序                     |
| `limit(long n)`      | 截取前 n 个元素                      |
| `skip(long n)`       | 跳过前 n 个元素                      |
| `flatMap()`          | 拆分扁平化流（集合中集合拆成单个流） |



## 3. 终结操作（触发执行，关闭流）

| 常用方法              | 作用                             |
| :-------------------- | :------------------------------- |
| `forEach(Consumer)`   | 遍历流中每个元素                 |
| `count()`             | 返回元素个数，返回 long          |
| `max(Comparator)`     | 获取最大值                       |
| `min(Comparator)`     | 获取最小值                       |
| `collect(Collectors)` | **收集流转集合 / Map**（最常用） |
| `anyMatch(条件)`      | 任意一个满足条件 → 返回 boolean  |
| `allMatch(条件)`      | 全部满足条件 → 返回 boolean      |
| `noneMatch(条件)`     | 全部不满足条件 → 返回 boolean    |
| `findFirst()`         | 获取第一个元素                   |
| `findAny()`           | 获取任意一个元素（并行流常用）   |



## 4. Collectors 收集器常用 API

| 方法                      | 作用                        |
| :------------------------ | :-------------------------- |
| `Collectors.toList()`     | 收集为 List 集合            |
| `Collectors.toSet()`      | 收集为 Set 集合（自动去重） |
| `Collectors.toMap(k,v)`   | 收集为 Map 集合             |
| `Collectors.counting()`   | 统计元素个数                |
| `Collectors.groupingBy()` | **分组**（按指定规则分组）  |
| `Collectors.joining()`    | 拼接字符串                  |

# 八、IO 流常用 API 总

## 1. IO 流四大抽象父类

| 抽象父类       | 分类       | 作用                                 |
| :------------- | :--------- | :----------------------------------- |
| `InputStream`  | 字节输入流 | 读文件（一切文件：文本、图片、视频） |
| `OutputStream` | 字节输出流 | 写文件                               |
| `Reader`       | 字符输入流 | 只读**文本**文件                     |
| `Writer`       | 字符输出流 | 只写**文本**文件                     |

> 字节流：万能流；字符流：只处理文本



## 2. 文件节点流（基础流）

### 2.1 字节文件流

| 类名               | 常用构造 / 方法                                              | 作用                                     |
| :----------------- | :----------------------------------------------------------- | :--------------------------------------- |
| `FileInputStream`  | `new FileInputStream(路径)`                                  | 从文件**读取**字节数据                   |
| `FileOutputStream` | `new FileOutputStream(路径)``new FileOutputStream(路径,true)` | 向文件**写入**字节加 true 为**追加写入** |

### 2.2 字符文件流

| 类名         | 常用构造 / 方法             | 作用                  |
| :----------- | :-------------------------- | :-------------------- |
| `FileReader` | `new FileReader(路径)`      | 读取文本文件字符      |
| `FileWriter` | `new FileWriter(路径,true)` | 写文本文件，true 追加 |



## 3. 缓冲流（高效，开发常用）

### 3.1 字节缓冲流

| 类名                   | 作用             |
| :--------------------- | :--------------- |
| `BufferedInputStream`  | 高效字节输入缓冲 |
| `BufferedOutputStream` | 高效字节输出缓冲 |

### 3.2 字符缓冲流（重点）

| 类名             | 特有常用方法 | 作用                       |
| :--------------- | :----------- | :------------------------- |
| `BufferedReader` | `readLine()` | **读取一行文本**（最常用） |
| `BufferedWriter` | `newLine()`  | **换行写入**               |



## 4. 转换流（字节流转字符流、指定编码）

| 类名                 | 作用                            |
| :------------------- | :------------------------------ |
| `InputStreamReader`  | 字节输入 → 字符输入，可指定编码 |
| `OutputStreamWriter` | 字节输出 → 字符输出，可指定编码 |



## 5. 打印流（便捷输出）

| 类名          | 常用方法              | 作用                             |
| :------------ | :-------------------- | :------------------------------- |
| `PrintStream` | `print() / println()` | 字节打印流，控制台 / 文件输出    |
| `PrintWriter` | `print() / println()` | 字符打印流，自动刷新、便捷写文本 |



## 6. 序列化流（对象读写）

| 类名                 | 常用方法            | 作用                             |
| :------------------- | :------------------ | :------------------------------- |
| `ObjectOutputStream` | `writeObject(对象)` | **把对象写入文件**（序列化）     |
| `ObjectInputStream`  | `readObject()`      | **读取文件中的对象**（反序列化） |

> 实体类需实现 `Serializable` 接口



## 7. IO 通用核心方法（所有流通用）

### 1.1 输入流通用

| 方法                  | 作用                   |
| :-------------------- | :--------------------- |
| `read()`              | 读 1 个字节 / 字符     |
| `read(byte[]/char[])` | 读入数组，返回读取长度 |
| `close()`             | 关闭流，释放资源       |



### 7.2 输出流通用

| 方法                    | 作用                 |
| :---------------------- | :------------------- |
| `write(int b)`          | 写出单个字节 / 字符  |
| `write(数组)`           | 写出整个数组         |
| `write(数组,起始,长度)` | 写出数组指定区间     |
| `flush()`               | 刷新缓冲区，强制写出 |
| `close()`               | 关闭流               |



## 8. IO 流选择口诀

1. **文本文件** → 优先用 **字符缓冲流 BufferedReader / BufferedWriter**
2. **图片 / 视频 / 音频 / 任意文件** → 用 **字节流 / 字节缓冲流**
3. **需要指定编码** → 用 **转换流 InputStreamReader / OutputStreamWriter**
4. **读写对象** → **Object 流 + Serializable**
5. **快捷打印写入** → **PrintWriter**
