# 加密方法

### MD5 常用方法（最常用）

```java
// 直接对字符串做MD5，返回32位小写密文
public static String md5DigestAsHex(String str)

// 字节数组 → MD5 → 32位十六进制字符串
public static String md5DigestAsHex(byte[] bytes)

// 返回加密后的byte[]，不转十六进制
public static byte[] md5Digest(byte[] bytes)
```



#### SHA 系列（SHA1 / SHA256 / SHA512）

```java
// 字节数组SHA1 → 十六进制字符串
public static String sha1DigestAsHex(byte[] bytes)
public static byte[] sha1Digest(byte[] bytes)
    
// SHA256加密
public static String sha256DigestAsHex(byte[] bytes)
public static byte[] sha256Digest(byte[] bytes)
    
// SHA512加密    
public static String sha512DigestAsHex(byte[] bytes)
public static byte[] sha512Digest(byte[] bytes)
```

注意：**Spring 的 DigestUtils 没有直接提供字符串入参的 SHA 方法**，需要自己转byte[]

```
String res = DigestUtils.sha256DigestAsHex("123456".getBytes());
```
