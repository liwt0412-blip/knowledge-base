# 网络编程

### 网络编程

可以让设备中的程序与网络中的其他设备中的进行数据交互的技术(实现网络通信)

### 基本架构

#### CS架构

效率高,因为大部分的数据存在本地

Client客户端/Server服务端

#### BS架构

Browser浏览器/Server服务端

这两个架构都依赖网络编程

#### Java提供了哪些网络编程解决方案?

java.net.*包下提供了网络编程的解决方案

## 网络编程三要素

IP地址  设备在网络中的地址,是设备在网络中的唯一标识     找到想与之通信的计算机

端口  应用程序在设备中的唯一标识     找到端口

协议   连接和数据在网络中传输的规则      语言



DNS 域名解析服务器  存的是域名跟IP地址对应关系

#### ip地址 

全称 互联网协议地址,是分配给上网设备的唯一标识

目前 被广泛采用额IP地址形式有两种  IPv4   IPv6

ipV4  32位地址  2的32次方   点分十进制  主流方案 用完了
