# MySQL笔记

## 一、MySQL安装

**详细安装过程参照官方文档：[MySQL安装文档](../06.权威笔记/MySQL安装文档.pdf)**



## 二、初始化命令总结

以**管理员**身份运行cmd

```
# 验证环境变量是否添加成功，如果提示‘Can't connect toMySQL server on 'localhost'’则说明添加成功
mysql

# 初始化命令
mysqld --initalize-insecure

# 注册MySQL服务
mysqld -install

# 启动MySQL服务
net start mysql

# 停止MySQL服务
net stop mysql

# 修改默认账户密码
mysqladmin -u root password 1234

# 登录mysql
mysql -uroot -p1234

# 登录mysql（加参）
mysql -u用户名 -p密码 -h要连接的mysql服务器的ip地址(默认127.0.0.1) -P端口号(默认3306)

# 退出mysql
exit

# 卸载mysql
net stop mysql
mysqld -remove mysql
```



## 三、MySQL命令

### 1. DDL 数据定义语言（操作库、表结构）

| 功能作用       | 小写语法                                       |
| :------------- | :--------------------------------------------- |
| 创建数据库     | `create database 库名;`                        |
| 删除数据库     | `drop database 库名;`                          |
| 使用数据库     | `use 库名;`                                    |
| 查看所有数据库 | `show databases;`                              |
| 创建数据表     | `create table 表名(字段 类型,字段 类型);`      |
| 删除数据表     | `drop table 表名;`                             |
| 查看表结构     | `desc 表名;`                                   |
| 查看建表语句   | `show create table 表名;`                      |
| 添加表字段     | `alter table 表名 add 字段 类型;`              |
| 修改字段类型   | `alter table 表名 modify 字段 新类型;`         |
| 修改字段名     | `alter table 表名 rename column 旧名 to 新名;` |
| 删除表字段     | `alter table 表名 drop 字段名;`                |
| 修改表名       | `alter table 旧表名 rename to 新表名;`         |



### 2. DML 数据操纵语言（增删改表数据）

| 功能作用         | 小写语法                                         |
| :--------------- | :----------------------------------------------- |
| 插入指定字段数据 | `insert into 表名(字段1,字段2) values(值1,值2);` |
| 插入全字段数据   | `insert into 表名 values(所有值);`               |
| 修改表数据       | `update 表名 set 字段=值 where 条件;`            |
| 条件删除数据     | `delete from 表名 where 条件;`                   |
| 摧毁整张表       | `truncate table 表名;`                           |



### 3. DQL 数据查询语言（查询数据）

| 功能作用     | 小写语法                          |
| :----------- | :-------------------------------- |
| 查询所有字段 | `select * from 表名;`             |
| 查询指定字段 | `select 字段1,字段2 from 表名;`   |
| 条件查询     | `select * from 表名 where 条件;`  |
| 数据去重     | `select distinct 字段 from 表名;` |
| 字段起别名   | `select 字段 as 别名 from 表名;`  |
| 升序排序     | `order by 字段 asc`               |
| 降序排序     | `order by 字段 desc`              |
| 分页查询     | `limit 起始索引,条数`             |
| 统计总数     | `count()`                         |
| 求和         | `sum()`                           |
| 求平均值     | `avg()`                           |
| 最大值       | `max()`                           |
| 最小值       | `min()`                           |
| 分组查询     | `group by 字段 having 条件`       |
| 模糊查询     | `like '%关键词%'`                 |
| 区间范围查询 | `between 值1 and 值2`             |
| 多条件匹配   | `in(值1,值2)`                     |
| 判断为空     | `is null`                         |
| 判断不为空   | `is not null`                     |



### 4. DCL 数据控制语言（权限 + 事务）

| 功能作用       | 小写语法                                            |
| :------------- | :-------------------------------------------------- |
| 创建数据库用户 | `create user '用户名'@'主机' identified by '密码';` |
| 分配权限       | `grant 权限 on 库.表 to '用户'@'主机';`             |
| 撤销权限       | `revoke 权限 on 库.表 from '用户'@'主机';`          |
| 查看用户权限   | `show grants for '用户'@'主机';`                    |
| 删除数据库用户 | `drop user '用户名'@'主机';`                        |
| 手动开启事务   | `start transaction;`                                |
| 提交事务       | `commit;`                                           |
| 回滚事务       | `rollback;`                                         |
| 关闭自动提交   | `set autocommit=0;`                                 |
| 开启自动提交   | `set autocommit=1;`                                 |



## 四、约束

| 约束类型 | 作用描述                        | 关键字      |
| -------- | ------------------------------- | ----------- |
| 非空约束 | 限制字段值不能为 null           | not null    |
| 唯一约束 | 字段内数据唯一，不可重复        | unique      |
| 主键约束 | 行数据唯一标识，**非空 + 唯一** | primary key |
| 默认约束 | 未赋值时自动填入默认值          | default     |
| 外键约束 | 关联两张表，保障数据关联完整性  | foreign key |



## 五、数据类型

### 1. 数值类型

| 类型        | 大小    | 有符号 (SIGNED) 范围                                  | 无符号 (UNSIGNED) 范围                                    | 描述                |
| ----------- | ------- | ----------------------------------------------------- | --------------------------------------------------------- | ------------------- |
| tinyint     | 1 byte  | (-128, 127)                                           | (0, 255)                                                  | 小整数值            |
| smallint    | 2 bytes | (-32768, 32767)                                       | (0, 65535)                                                | 大整数值            |
| mediumint   | 3 bytes | (-8388608, 8388607)                                   | (0, 16777215)                                             | 大整数值            |
| int/integer | 4 bytes | (-2147483648, 2147483647)                             | (0, 4294967295)                                           | 大整数值            |
| bigint      | 8 bytes | (-2^63, 2^63-1)                                       | (0, 2^64-1)                                               | 极大整数值          |
| float       | 4 bytes | (-3.402823466 E+38, 3.402823466351 E+38)              | 0 和 (1.175494351 E-38, 3.402823466 E+38)                 | 单精度浮点数值      |
| double      | 8 bytes | (-1.7976931348623157 E+308, 1.7976931348623157 E+308) | 0 和 (2.2250738585072014 E-308, 1.7976931348623157 E+308) | 双精度浮点数值      |
| decimal     |         | 依赖于 M (精度) 和 D (标度) 的值                      | 依赖于 M (精度) 和 D (标度) 的值                          | 小数值 (精确定点数) |



### 2. 字符串类型

| 类型       | 大小                  | 描述                          |
| ---------- | --------------------- | ----------------------------- |
| char       | 0-255 bytes           | 定长字符串 (需要指定长度)     |
| varchar    | 0-65535 bytes         | 变长字符串 (需要指定长度)     |
| tinyblob   | 0-255 bytes           | 不超过 255 个字符的二进制数据 |
| tinytext   | 0-255 bytes           | 短文本字符串                  |
| blob       | 0-65 535 bytes        | 二进制形式的长文本数据        |
| text       | 0-65 535 bytes        | 长文本数据                    |
| mediumblob | 0-16 777 215 bytes    | 二进制形式的中等长度文本数据  |
| mediumtext | 0-16 777 215 bytes    | 中等长度文本数据              |
| longblob   | 0-4 294 967 295 bytes | 二进制形式的极大文本数据      |
| longtext   | 0-4 294 967 295 bytes | 极大文本数据                  |



### 3. 时间日期类型

| 类型      | 大小 (字节) | 范围                                       | 格式                | 描述                     |
| --------- | ----------- | ------------------------------------------ | ------------------- | ------------------------ |
| date      | 3           | 1000-01-01 至 9999-12-31                   | YYYY-MM-DD          | 日期值                   |
| time      | 3           | -838:59:59 至 838:59:59                    | HH:MM:SS            | 时间值或持续时间         |
| year      | 1           | 1901 至 2155                               | YYYY                | 年份值                   |
| datetime  | 8           | 1000-01-01 00:00:00 至 9999-12-31 23:59:59 | YYYY-MM-DD HH:MM:SS | 混合日期和时间值         |
| timestamp | 4           | 1970-01-01 00:00:01 至 2038-01-19 03:14:07 | YYYY-MM-DD HH:MM:SS | 混合日期和时间值，时间戳 |
