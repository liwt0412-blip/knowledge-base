# HTML，CSS，JS

### HTML

构建整个网页的框架，是超文本标记语言

超文本：超越文本，比文本更强大；可以插入图片/视频/音频/文字

标记语言：一对一对的标签语言，预定义标签（就是定义好了的标签）

标签分为四种：

##### 标题标签：h1-h6  

（数字越小，字体越大，加粗，数字越大即反之）

图片标签：img

src：指定路径

alt：图片无法加载时显示的文字

##### 表单标签：（提交数据给后端）

from-action-指定请求路径

method：

请求方式分为两种：get/post

get：请求参数，在浏览器url地址栏，动态拼接url，有大小限制，2kb，不安全；

post：请求参数在请求体中，没有大小限制，可以上传文件，相对安全

input：（输入框）

type（类型）：text/passwo/email/number

text（文本）/password（密码，密文展示）/email（邮箱，可以完成简单的表单校验）/number（数字款，只能输入数字）

##### select（下拉列表）：option/value/name

option（指定下拉元素）&  value(提交数据）

属性：name（用来提交数据）

textarea（文本域，就是一个大小可变的文本输入框）

button（按钮）：submit（提交表单）& reset(清空表单）)

##### 其他标签：table/audio/video/p/hr/br

table（表格）：thead/tbody/tr/th/td

thead（表头）：表示表头部分，不具备任何效果，语义化标签

tbody：表示表格数据部分

tr：定义行

th：定义表头单元格

td：定义行中单元格

audio：音频标签

video：视频标签

p：段落标签

hr：下划线标签

br：换行标签

#### css

美化图层样式的作用/层叠样式表

##### 引入方式：

行内标签：写在标签的style属性中（配合JavaScript使用）

内部样式：写在style标签中（可以写在页面任何位置，但通常约定写在head标签中）

外部样式：写在一个单独的.css文件中(需要通过link标签在网页中引入)



##### 选择器：用来找到元素

###### 元素选择器：元素名称{......} 

例句：h1{.......}   //选择页面上所有的<h1>标签

###### 类选择器：.class属性值{......}

例句：.cls{......}   //选择页面上所有的class属性为cls的标签

###### id选择器：#id属性值{......}

例句：#hid{......}  //选择页面上id属性为hid的标签

//不太常用的

分组选择器：选择器1选择器2{.......}

例句：h1，h2{.......}  //选择页面上所有的<h1>和<h2>标签

属性选择器（有两个）：

元素名称[属性]{......}

例句：input[type]{......}  //选择页面上有type属性的<input>标签

元素名称[属性名 = “值”]{......}

例句：input[type = “text”]{......}  //选择页面上type属性为text的<input>标签

后代选择器：

元素1 元素2{......}

例句：form input{...... }   //选择<form>标签内的所有<input>标签



#### JavaScript

增加用户与页面的交互

###### 引入方式：

内部脚本：将js代码定义在HTML页面中，JavaScript代码必须位于<script></script>标签之间

外部脚本：将js代码定义在外部js文件中，然后引入到HTML页面中

###### 基础语法：

定义变量：

弱数据类型语言，定义变量不需要指定类型

定义变量规则与java相同

let定义局部变量

const定义常量

数据类型：

JavaScript的数据类型分为：基本数据类型和引用数据类型（对象）

###### 基本数据类型：

number：数字（整数，小数）

Boolean：布尔（true，false）

null：对象为空（JavaScript似乎大小写敏感的，因此null,Null,NULL是完全不同的）

underfined：当声明的变量未初始化时，该变量的默认值是underfined

string：字符串，单引号，双引号，反引号皆可，推荐使用单引号

###### 数据引用类型：

数组，自定义对象，等等

###### 函数：

函数（function）是被设计用来执行特定任务的代码块，方便程序的封装复用

也就是java中的方法

三种定义方式：（调用方法：直接通过方法名调用即可）

function add（参数1，参数2...）{

​        //要执行的代码块

}

let add = function（参数1，参数2）{

​        //要执行的代码块     （函数表达式）

}

let add = （参数1，参数2）=>{

​        //要执行的代码块     （箭头函数）

}

注意：1.函数不需要写返回值类型，直接在方法中return结果即可

2.参数也不需要写数据类型

3.调用方法时，实参和形参不是不匹配的，通常都匹配传递参数

4.函数没有重载，后面定义相同名字的方法，会直接覆盖前面定义的

###### 事件监听：

事件：HTML事件时发生在HTML元素上的“事情”

比如：按钮被点击，鼠标移动到元素上，按下键盘按键等

事件监听：JavaScript可以在事件除法时，就立即调用一个函数做出响应，也成为事件绑定或注册事件

简单理解：就是某个元素触发某个操作，则会执行某个方法

例句：

<input id = "btn"  type = "button"  value = "点我触发事件">

<script>
    document.querySelector('#btn').addEvenListener('click',()=>{
        alert('点击试试')；
    })
</script>

<input id = "btn"  type = "button"  value = "点我触发事件">

<script>
    document.querySelector('#btn').onclick = function(){
        alert('点击试试')；
    }
</script>
