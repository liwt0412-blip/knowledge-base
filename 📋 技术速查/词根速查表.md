# 印欧语音变规律 & 常用词根速查表

> 格式：PIE 原始印欧语词根 → 日耳曼路径 → 英语原生词 | 拉丁路径 → 英语借词
> 音变规律（格里姆定律）：p→f, t→th, k→h, b→p, d→t, g→k, bh→b, dh→d, gh→g

### 音变串联实战案例

#### 案例1：`configuration`

`con` + `figur` + `ation`

- `figur` → `figure` 形状/人物 → 来自PIE *dheigh- "揉捏/塑造"
- 日耳曼路径：同一词根 → `dough` 面团（被揉捏的东西）
- 📌 d↔f 不对应... 等一下，这里是拉丁 fig- 对应日耳曼 d- (dough)
  - PIE *dh → 拉丁 f, 日耳曼 d  ✔ 
  - *dheigh- → 拉丁 fig- (形状), 日耳曼 d- (dough 面团)
  - 所以 figure(形状) 和 dough(面团) 是同源的！都是"被揉捏出来的东西"

#### 案例2：`generate` / `generic` / `gene`

- PIE *gen- "生育/产生/种族"
- 日耳曼路径 → kin 亲属, kind 种类/善良, king 国王(一族之长)
- 拉丁路径 → **gen**erate 产生, **gen**etic 基因的, **gen**eral 普遍的
- 📌 注意：gener- 中的 g 来自 PIE *g, 日耳曼对应 k, 所以 kin↔generate

#### 案例3：`transmit`

`trans`("跨越") + `mit`("发送")

- PIE *mei- "变化/交换/移动"
- 拉丁路径 → **mit**ter 发送, **miss**ion 任务, **miss**ile 导弹
- 📌 mit-/miss- 来自拉丁 mittere，本义是"放出/投掷"

#### 案例4：`transaction`

`trans`("跨越") + `act`("做") + `ion`

- PIE *ag- "驱动/做"
- 拉丁路径 → **ag**ent 代理, **act**ion 行为, **ag**ile 敏捷, ex**am** (exigere 驱出→检查)
- 日耳曼路径？→ 这个在日耳曼中不明显，可能跟 drive 有关...跳过

#### 案例5：`interface`

`inter`("之间") + `face`("面")

- PIE *dhe- "放置/做"
- 拉丁路径 → **fa**ce 脸(原义"外观/形状"), **fa**cade 外表, sur**fa**ce 表面
- facere = 做 → factum = 做过的事 → face = "做出来的样子"即"面貌"
- 同根：**fact**ory, **feat**ure, **fash**ion

#### 案例6：`abstract`

`abs`("离开") + `tract`("拉")

- PIE *tragh- "拉/拖"
- 拉丁路径 → **tract**or 拖拉机, at**tract** 吸引, con**tract** 合同(拉到一起), ex**tract** 提取, re**tract** 撤回
- 日耳曼路径 → **drag** 拖拽(!)，**draw** 拉(!)
- 📌 abstract (拉开→抽象) ↔ drag (拖拽)，同一个原始词根
