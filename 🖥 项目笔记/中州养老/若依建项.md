# 若依框架的快速开发

# 一、若依搭建初始化

### 1. 软件版本准备

| 官方推荐                  | 本机版本                                  |
| ------------------------- | ----------------------------------------- |
| JDK >= 1.8（推荐1.8版本） | JDK 11、JDK 17                            |
| Maven >=3.0               | 不需要安装，直接使用IDEA中自带的Maven即可 |
| Nodejs >= 12（Vue 2）     | Nodejs 18.20.4.0                          |
| Nodejs >= 18 （Vue 3）    | Nodejs 18.20.4.0                          |
| MySQL >= 5.7.0            | MySQL 8                                   |
| Redis >= 3.0              | Redis 7                                   |

**vue2与vue3的区别：**

1. vue2需要使用Nodejs 12以上版本，vue3需要使用Nodejs 18以上版本

2. vue2是选项式，vue3是组合式

3. vue3支持人工智能

   

### 2. 下载项目模板源码，包名全局替换

**2.1 官网下载地址：**[https://gitee.com/y_project/RuoYi-Vue ](https://gitee.com/y_project/RuoYi-Vue)

推荐下载3.9.1版本或3.9.2版本，这两个版本有很大的不同：

3.9.1版本 要使用jdk 11版本，3.9.2版本要使用 jdk 17及以上版本，页面样式、布局全新风格，并且支持人工智能。

# 二、若依初建项

1. 在“系统管理”——“菜单管理”里新建菜单
2. 在mysql里新建一张空表，设置好字段，但不用设置值
3. 在“系统工具”——“代码生成”里导入设置好的那张空表，根据业务需求分别设置好基本信息、字段信息和生成信息
4. 提交后点击“生成”，就可以下载对应的代码包了
5. 将代码包里java文件夹导入后端，vue文件夹导入前端，重启项目就可以在页面上看到项目了。
