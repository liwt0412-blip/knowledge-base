```text

```

## Hermes 相关入口

Hermes 记忆备份目录：

```text
D:\KnowledgeBase\小帅的知识库\🤖 AI Agent\Hermes记忆备份
```

读取时机：

- 需要追溯 Hermes 曾经记住过什么。
- 压缩、删除或迁移 Hermes memory 前。
- 多 Agent 记忆出现冲突，需要查历史来源时。

规则：

- Hermes 备份是历史材料，不默认作为当前事实。
- 迁移 Hermes 记忆前，先分流到长期上下文、当前主线、知识库地图或专题笔记。
- 备份中如出现 token、密钥、账号等敏感信息，应先打码，不应进入长期上下文。

## 工具环境类记忆

工具环境配置不要写入全局长期上下文，除非会反复影响 Agent 操作。

目前保留为全局规则的工具记忆：

- Hermes 环境禁止用 `pip` 安装或升级包，优先用 `uv`，避免破坏 Hermes 环境。
- Codex CLI 使用代理时，通常需要手动设置 `HTTP_PROXY` 和 `HTTPS_PROXY`。

其他版本号、端口号、临时路径、一次性报错，默认保留在对应工具笔记或历史备份里。
