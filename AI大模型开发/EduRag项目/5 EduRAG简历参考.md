# 05-EduRAG简历参考

# 项目名称：基于RAG的教育领域智能问答系统（EduRAG）

**项目时间**：2025.03 -- 2025.08\
**技术栈**：Python, Milvus, BERT, Sentence-BERT,BGE-M3, Qwen2.5-7B, LoRA, RagAS, OCR, FastAPI

**项目背景**

针对教培行业高频、重复性咨询问题（如课程安排、退费政策、学习规划等），传统FAQ匹配准确率低、知识更新滞后，难以满足用户实时精准答疑需求。为此设计并实现了一套融合结构化数据检索与RAG技术的智能问答系统------**EduRAG**，显著提升回答准确率与用户体验。

**主要职责**

构建RAG知识库，整理并清洗PDF/Word/PPT等文档资料，累计构建 **50万+条数据**；

文档预处理：实现文本加载、去噪清洗、智能分块（按语义边界）、多模态内容处理

搭建 **Milvus 向量数据库**，设计多集合（3个collections）存储策略，支持课程、政策、服务等不同类型知识的隔离管理与高效检索；

基于 **BERT 意图识别模块**，支持4种意图分类如：（课程咨询、退费政策、学习规划、通用问题），F1-score达 **90%**；

完整RAG流程：根据意图路由至对应检索路径，结合检索结果拼接Prompt，基于 **开源Qwen2.5-7B模型生成自然语言回答**，支持上下文感知与格式化输出。

**项目优化与技术创新**

**多模态文档处理增强**\
针对PDF/PPT中图片、表格等内容，引入 OCR工具（PaddleOCR）与多模态大模型提取图文信息，提升知识覆盖完整性。

**文档索引与检索优化**

构建 **多级索引机制**（支持父块/子块切分策略）；

实现 **稀疏向量 与 稠密向量 混合检索**，通过加权融合提升召回准确率；

引入 **（Rerank）模块**，对Top-k结果进行精细化排序，Top-5相关性提升35%。

**模型微调与生成优化**

对 **Qwen2.5-7B模型进行LoRA微调**，适配教育领域术语与回答风格，显著提升回答专业性与流畅度；

设计结构化Prompt模板，控制生成格式，减少幻觉与冗余。

**系统评估体系建设**

使用 RagAS 自动评估框架对RAG全流程进行量化评测，关键指标：

答案相关性（Answer Relevance）≥ 0.89

忠实性（Faithfulness）≥ 0.9

构建 **1000条人工标注测试集**，整体准确率（Accuracy）达 **90%**，具备良好泛化能力。

**项目成果**

系统上线后自动应答率超 **80%**，人工客服压力降低约50%；

回答准确率与用户满意度显著提升，获内部创新项目认可；

支持知识库动态更新，具备高可扩展性与私有化部署能力。
