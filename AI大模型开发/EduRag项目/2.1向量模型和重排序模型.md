# 向量模型和重排序模型

## 1. 向量模型

在模型开放平台中，**huggingface**和**魔搭**中开源了很多的优秀的模型，有一些嵌入模型，比三方平台提供的模型还要优秀，我们以魔搭中的**BAAI/bge-m3**模型为例

### 1.1 BAAI/bge-m3

模型介绍

**[该类型的内容暂不支持下载]**

**特点：**

支持多种语言嵌入，超过100种语言

长文本处理，最高可支持8192个token

多功能检索，支持**稠密向量**嵌入和**稀疏检索**(词频检索)

模型维度，可支持1024维

开源免费，可靠实用

### 1.2 下载模型

在下载之前，先安装ModelScope

```bash
pip install modelscope

```

下载完整库

```bash
modelscope download \--model BAAI/bge-m3

```

指定目录下载或下载单个文件

```bash
modelscope download \--model BAAI/bge-m3 README.md \--local_dir ./dir

```

### 1.3 使用模型

安装依赖：

```bash
pip install -U FlagEmbedding

```

下载完成之后，拷贝代码中目录中

代码示例：

```python
*\# 导入 BGE-M3 嵌入函数，用于生成文档和查询的向量表示*
import os

from milvus_model.hybrid import BGEM3EmbeddingFunction
from base.config import Config
from base.logger import logger
import sys

local_path = os.path.abspath(os.path.dirname(__file__))
rag_qa_path = os.path.abspath(os.path.dirname(local_path))
sys.path.insert(0, rag_qa_path)

conf = Config()


def create_vector_store():
*\# 自动检测设备：优先使用 CUDA，否则使用 CPU*
import torch
if torch.cuda.is_available():
device = 'cuda'
elif hasattr(torch.backends, 'mps') and torch.backends.mps.is_available():
device = 'mps'
else:
device = 'cpu'
logger.info(f"使用设备: {device}")

*\# 加载本地模型 bge-m3*
bge_m3_model_path = os.path.join(rag_qa_path, 'models', 'bge-m3')
*\# 加载*
embedding_function = BGEM3EmbeddingFunction(model_name_or_path=bge_m3_model_path, use_fp16=True, device=device)
'''
BGE-M3 是多向量嵌入模型，常见有两种：
dense（密集向量）：用于语义相似度检索
sparse（稀疏向量）：用于词汇匹配检索
'''
*\# 使用bge-m3嵌入生成文档的嵌入*
texts = ['今天北京天气挺好的','北京天气是晴天','上海水上交通堵了']
embeddings = embedding_function(texts)

dense_dim = embedding_function.dim["dense"]
print(f"嵌入向量的维度：{dense_dim}")

print(embeddings)

for i,text in enumerate(texts):
logger.info(f"文档 {i} 的嵌入向量：{embeddings['dense'][i]}")
logger.info(f"文档 {i} 的稀疏向量：{embeddings['sparse'][[i]]}")


if __name__ == "__main__":
create_vector_store()

```

## 2. 重排序模型

### 2.1 bge-reranker-large

BGE-Reranker-Large 是一个专门用于重排序（Re-ranking） 的深度学习模型，由北京智源人工智能研究院开发。它基于 Transformer 架构，专门优化了查询-**文档对**的相关性评分任务。

### 2.2 模型下载

```bash
modelscope download \--model BAAI/bge-reranker-large \--local_dir ./bge-reranker-large

```

### 2.3 模型使用

```python
from sentence_transformers import CrossEncoder
model1 = CrossEncoder(model_name="D:/ai_workspace/code/428/edu_rag_project/rag_qa/models/bge-reranker-large")

pairs = [['what is panda?', "what is panda?"],
['what is panda?', 'hi'],
['what is panda?', 'The giant panda '],
]

print(model1.predict(pairs)) \# [0.999556 0.003653 0.3362122]

```
