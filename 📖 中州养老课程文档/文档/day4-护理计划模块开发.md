6- 入住办理-前置业务
	1- 楼层、房间、床位、房型业务介绍：老人入住要选床位（床位属于某个房间，房间属于某个楼层，房间还有房间类型）
	2- 表关系
		floor -> 多个room -> 多个 bed
		room_type -> 多个room
	3- 编辑房间后，新建房间，报ID重复Bug（前端Bug） 
		1- 原因：编辑时，反显数据将ID记录在表单上了，新增时会携带ID到后端，导致重复
		2- 解决：最终新增时，将房间ID设置为null （views/nursing/floor/index.vue）
				handleRoomAddSubmit() -> params 定义下面新增代码：params.id = null;
	4- 编辑床位后，新建床位，报ID重复Bug（前端Bug）  
		1- 原因：编辑时，反显数据将ID记录在表单上了，新增时会携带ID到后端，导致重复
		2- 解决：最终新增时，将床位ID设置为null （views/nursing/floor/index.vue）
				handleBedAddSubmit() -> params 定义下面新增代码：params.id = null;
