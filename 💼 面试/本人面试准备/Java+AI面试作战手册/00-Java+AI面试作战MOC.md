## 当前资料判断

当前作战手册适合的岗位：

- Java 后端开发。
- Java + AI 应用开发。
- 企业内部 AI 助手、知识库问答、经营分析、业务系统智能化岗位。

不适合直接对标的岗位：

- 大模型训练算法岗。
- 强算法竞赛岗。
- 纯基础架构内核岗。
- 要求完整 AI 平台从零设计的架构师岗。

当前核心优势：

- 有明确主项目，不是散点八股。
- AI 讲法能落到 Java 业务系统。
- 有真实 Bug、接口联调、团队协作材料。
- 有职责边界和红线，面试不容易讲过头。

当前主要短板：

- Java 基础仍然要高频刷。
- 项目真实指标不要乱编。
- AI 复杂概念只在追问时展开，不要主动堆术语。

最终面试目标：

> 让面试官相信你是一个能进项目、能写业务、能排问题、能把 AI 能力接进 Java 系统的后端开发。
