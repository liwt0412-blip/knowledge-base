## 三、Java 技术面试题

### Spring Boot 篇

#### 1. @SpringBootApplication 的原理？

组合注解：`@SpringBootConfiguration` + `@EnableAutoConfiguration` + `@ComponentScan`

自动配置原理：`@EnableAutoConfiguration` → `AutoConfigurationImportSelector` → 加载 `META-INF/spring/...imports` 文件 → 配合 `@ConditionalOnClass` 等条件注解按需装配。

#### 2. @Autowired vs 构造器注入？

| 对比 | 字段注入 | 构造器注入 |
|------|---------|-----------|
| 写法 | `@Autowired private UserService us;` | `private final UserService us;` + 构造器 |
| 不可变性 | ❌ | ✅ final |
| 测试 | ❌ 需反射 | ✅ 直接new |
| Spring推荐 | ❌ | ✅ |

#### 3. @RestController vs @Controller？

`@RestController = @Controller + @ResponseBody`，直接返回 JSON 不经过视图解析。

#### 4. RESTful 设计原则？

- 资源用名词复数：`/users` `/courses`
- HTTP 方法对应 CRUD：GET查 POST增 PUT改 DELETE删
- 状态码：200/201/204/400/404/500
- 无状态 + JWT 鉴权

### MyBatis 篇

#### 1. #{} vs ${}？

| | `#{}` | `${}` |
|---|-------|-------|
| 处理 | 预编译占位符，生成 `?` | 字符串直接拼接 |
| 安全 | ✅ 防SQL注入 | ❌ 有注入风险 |
| 场景 | 条件值 | 动态表名/列名 |

#### 2. 一级缓存 vs 二级缓存？

- **一级缓存**（SqlSession级别）：默认开启，同Session共享
- **二级缓存**（namespace级别）：需手动开，实际不推荐，不如用Redis

#### 3. 动态 SQL 标签？

`<if>` `<where>` `<set>` `<foreach>` `<choose>/<when>/<otherwise>`

#### 4. resultType vs resultMap？

- resultType：字段名和列名一致时自动映射
- resultMap：自定义映射 + 支持association/collection一对多

#### 5. PageHelper 分页原理？

`startPage()` → MyBatis拦截器拦截查询 → 自动加 COUNT + LIMIT → PageInfo 封装 → 清理 ThreadLocal

### Lombok 篇

#### @Data 包含哪些？

`@Getter + @Setter + @ToString + @EqualsAndHashCode + @RequiredArgsConstructor`

#### @Builder + @Data 要注意什么？

必须同时加：`@Data` + `@Builder` + `@NoArgsConstructor` + `@AllArgsConstructor`
缺 `@AllArgsConstructor` → Builder 编译报错
缺 `@NoArgsConstructor` → Jackson 反序列化失败

### 安全篇

#### 为什么不能只用 MD5？

碰撞攻击 + 彩虹表 + 速度太快。
正确做法：bcrypt/Argon2 + 随机盐。至少 MD5 + salt。

### 异常处理篇

```java
@RestControllerAdvice
public class GlobleExceptionHandler {
    @ExceptionHandler(BusinessException.class)
    public Result handleBusiness(BusinessException e) {
        return Result.error(e.getMessage());
    }
    @ExceptionHandler(Exception.class)
    public Result handleOther(Exception e) {
        log.error("系统异常", e);
        return Result.error("服务器忙，请稍后再试");
    }
}
```

### 分页篇

| 方式 | 说明 |
|------|------|
| LIMIT offset, size | 简单，深分页性能差 |
| PageHelper | 方便，适合中小项目 |
| 游标分页 | 性能好，不适合随机跳页 |

深分页优化：先查主键再 JOIN。

### 阿里云 OSS 篇

三步：配置 → `@ConfigurationProperties` 绑定 → 创建 OSSClient → putObject → 返回 URL

安全注意：限制大小、校验类型、UUID重命名、STS临时凭证。
