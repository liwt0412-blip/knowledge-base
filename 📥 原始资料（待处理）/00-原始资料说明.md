# 📥 原始资料（待处理）

> [!abstract] 用途
> 这里只保存从网络或外部收集、尚未仔细阅读、验证或提炼的**原始资料**。它不是正式笔记、学习结论，也不是待办清单。

## 可以放什么

- 网页文章、公众号文章、博客、文档的本地副本
- PDF、电子书、课程讲义、视频字幕或转写稿
- 项目需求、截图、产品资料及其他外部参考材料

## 不放什么

- 已理解后的个人结论：写入对应的正式专题笔记
- 可执行事项：写入当前任务或项目记录，不在这里建立行动卡
- 已形成稳定结论的内容：归入 `☕ Java笔记`、`🤖 AI Agent`、`🗂 项目笔记` 等对应模块

## 收集时的最小记录

每份 Markdown 资料在顶部尽量保留：

- 来源链接或来源说明
- 收集日期
- 收集原因（用一句话说明即可）

文件名建议：`YYYY-MM-DD-主题-来源.md`。原文件无法改名时，至少在同名说明笔记中补齐来源信息。

## 处理后的去向

1. **提炼**：把经阅读、核验后的结论写入正式笔记，并从这里链接过去。
2. **归档**：资料有保留价值但暂不处理时，保留原件并注明状态。
3. **移除**：确认重复、失效或无价值时再删除；不确定时优先保留。

> [!warning] Agent 规则
> Agent 不得把本目录的内容当作已掌握的知识、项目事实或面试口径。引用其中资料前，应先阅读、核验，并将可靠结论沉淀到正式笔记。
